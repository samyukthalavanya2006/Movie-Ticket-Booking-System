# Movie Ticket Booking System

## Project Overview
This project is a console-based Movie Ticket Booking System developed using Python and MySQL.

The system allows users to:
- View movie show timings
- Select movie screens
- Enter customer details
- Generate customer IDs
- Book movie tickets
- Calculate ticket prices
- Store booking details in a MySQL database

This project was created as a mini-project to demonstrate Python programming and database connectivity using MySQL.

---

# Technologies Used
- Python
- MySQL
- MySQL Connector

---

# Features
- Database creation using Python
- MySQL table creation
- Movie show selection
- Customer detail management
- Random customer ID generation
- Ticket booking system
- Multiple ticket categories
- Automatic bill calculation
- Payment simulation
- Database record storage

- Ticket Price: Rs.850
